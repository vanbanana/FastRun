这里的关键点（给全栈工程师的 Tips）
Prompt 3 中的 subprocess： 很多 AI 可能会用 os.system。你要注意，os.system 会在这个黑框框里阻塞，如果你关掉黑框，主程序可能也会挂。subprocess.Popen 才是正解。

Prompt 4 中的 lambda 陷阱： 在 Python 的循环中绑定事件（比如循环生成按钮）时，如果直接写 lambda: self.launch(path)，所有按钮都会指向最后一个路径。一定要用 lambda p=path: self.launch(p) 或者 functools.partial。如果 AI 写错了，你要能指出来。
